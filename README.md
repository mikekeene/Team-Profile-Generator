# Employee Tracker
## Description
The assignment is to build a command-line application from scratch to manage a company's employee database, using Node.js, Inquirer, and MySQL.
## Table of Contents 
* [Installation](#Installation)
* [License](#License)

## Installation
* Clone repository
* Install mysql 
* npm init 
* npm i (install packages)
* mysql -u root -p 
* source all db files
* npm start (start app)
